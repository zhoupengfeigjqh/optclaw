# 可用技能清单

## 🔧 核心系统技能
| 技能 | 描述 |
|------|------|
| **bootstrap** | 个性化 AI 身份设定（创建/更新 SOUL.md） |
| **find-skills** | 发现和安装新技能 |

## 🌐 研究与分析
| 技能 | 描述 |
|------|------|
| **deep-research** | 多角度深入研究（替代普通搜索） |
| **github-deep-research** | GitHub 仓库深度分析（时间线、指标等） |
| **systematic-literature-review** | 系统性文献综述（arXiv 论文搜索与 APA/IEEE 格式） |
| **consulting-analysis** | 专业咨询级研究报告（市场、品牌、金融分析等） |

## 📊 数据处理
| 技能 | 描述 |
|------|------|
| **data-analysis** | Excel/CSV 数据分析（透视表、SQL 查询等） |
| **chart-visualization** | 26 种图表智能生成 |

## 💻 开发与部署
| 技能 | 描述 |
|------|------|
| **code-documentation** | 代码文档生成（README、API 文档、架构文档等） |
| **vercel-deploy** | 一键部署到 Vercel |
